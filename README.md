# AiMenu POS

Tablet-first Expo / React Native app for restaurant operators — accept
reservations, manage orders, and run your floor from an iPad or the web.

## Quick start

```bash
cd aimenu-pos
npm install
cp .env.example .env         # set EXPO_PUBLIC_API_URL if needed
npm run web                  # browser preview
npm run ios                  # iPad simulator
npm run android
```

Set `EXPO_PUBLIC_API_URL` to override the default backend
(`https://admin.aimenu.ge`).

## Features

- **Login** — JWT auth against `/api/v1/auth/login/` (refresh via `/api/v1/auth/token/refresh/`).
- **Dashboard** — today's reservations with live stats (waiting / confirmed /
  today total). Tablet/desktop renders a 2–3 column grid matching the Figma
  design; mobile collapses to a single column.
- **Reservation card** — accept/reject inline, or open the full detail modal
  for extra context (special requests, history, no-show, cancel, etc.).
- **Orders** — Kanban board (pending → confirmed → preparing → ready → served).
  Auto-refresh every 10 s. Tap to advance through states.
- **i18n** — Georgian (default) and English, toggled from the top bar or
  settings. Locale is persisted.
- **Web** — `npm run build:web` exports a static bundle (served from `dist/`).

## Stack

- Expo 54 + Expo Router (typed routes)
- TanStack Query for data + polling
- AsyncStorage for token + locale persistence
- Axios with single-flight token refresh
- TypeScript strict mode
- `@gordela/api-generator` for regenerating the typed API client

## API client

The typed client in `src/api/generated/` is autogenerated from the backend's
drf-spectacular schema. Regenerate with:

```bash
npm run generate:api
```

Variables (`API_URL`, `SWAGGER_PATH`, `API_NAMESPACE`, `OUTPUT_DIR`) are read
from `.env` — same pattern as the customer frontend.

## Scripts

| Script                 | Purpose                          |
| ---------------------- | -------------------------------- |
| `npm run start`        | Start Metro dev server           |
| `npm run web`          | Start Metro + open the browser   |
| `npm run ios`          | Start iOS simulator (iPad)       |
| `npm run android`      | Start Android emulator           |
| `npm run build:web`    | Export static web bundle         |
| `npm run type-check`   | `tsc --noEmit`                   |
| `npm run generate:api` | Regenerate the typed API client  |
